Core Capabilities

Production-grade autonomous driving features validated in simulation

🛣️

Lane Following

Regulated pure pursuit controller with adaptive lookahead. Handles multi-lane scenarios at variable speeds (0-8 m/s).

  • Adaptive lookahead distance
  • Multi-lane support
  • Curve handling
  • Speed optimization
🚧

Obstacle Detection

Lidar-based 2D costmap with emergency braking. 100% safety margin with collision avoidance.

  • Real-time lidar fusion
  • Dynamic obstacle detection
  • Emergency braking
  • Inflation radius safety
🚦

Traffic Lights

Automatic traffic light detection and enforcement. Stops on red, resumes on green with full rule compliance.

  • Light state detection
  • Automatic stop/go
  • Rule compliance
  • Human-safe behavior
📡

Multi-Sensor Fusion

3× RGB cameras, 64-ray lidar, IMU, and GNSS. Time-synchronized sensor data for robust perception.

  • 3x HD cameras
  • High-res lidar
  • IMU + GNSS
  • Time-synced data
⚡

Real-Time Control

20 Hz deterministic control loop. 40-50ms end-to-end latency with 30-40% CPU efficiency.

  • 20 Hz loop rate
  • 50ms latency
  • Efficient processing
  • Deterministic timing
🎨

Live Visualization

Real-time RViz costmap views and debugging. Live sensor feeds with performance monitoring.

  • RViz costmaps
  • Path visualization
  • Trajectory tracking
  • Performance metrics

System Architecture

Professional 4-layer design for hardware-agnostic autonomous driving

🖥️ Layer 4: Application

CARLA UE5 simulator, Python bridge, custom ROS2 nodes, visualization tools

🔄 Layer 3: Navigation2

Costmap 2D, path planning, pure pursuit controller, safety layer, behavior trees

🌐 Layer 2: ROS2 Middleware

ROS2 Humble, DDS messaging, TF2 transforms, standard message types, time sync

⚙️ Layer 1: Hardware

NVIDIA GPU, cameras, lidar, CPU, memory, Ubuntu 22.04, CAN bus integration
